# Python-ATM-Simulation
A Python-based ATM simulation featuring user balance management, dynamic transaction loops, and logic-based error handling.
## Öne Çıkan Özellikler
- **Sürekli İşlem Döngüsü:** `while True` yapısı ile kullanıcının çıkış yapana kadar kesintisiz işlem yapabilmesi sağlanmıştır.
- **Bakiye Yönetimi:** Gerçek zamanlı para yatırma ve çekme işlemleriyle değişken (variable) yönetimi.
- **Güvenlik Kontrolü:** Çekilmek istenen tutarın mevcut bakiyeyi aşması durumunda işlemin reddedilmesi ve kullanıcıya uyarı verilmesi.
- **Çıkış Mekanizması:** 'q' tuşu ile programın güvenli bir şekilde sonlandırılması.

## Teknik Detaylar
- **Dil:** Python 3.x
- **Kullanılan Yapılar:** - `while` döngüleri (Loops)
  - `if-elif-else` koşullu ifadeleri (Conditionals)
  - F-string / .format() metin biçimlendirme

## Kullanım Talimatı
1. Projeyi bilgisayarınıza indirin.
2. `atm_makinasi.py` dosyasını terminal üzerinden çalıştırın.
3. Menüdeki numaraları (1, 2, 3) kullanarak işlemlerinizi gerçekleştirin.
4. Çıkmak için 'q' tuşuna basın.
